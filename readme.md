# PUTSB-PSNTSB (Pag Utang Tawag Sakin Boss, Pag Singil na Tawag Sakin Bantot)

## The Neural Debt Matrix

A specialized financial node tracker designed for high-stakes debt management in the sprawl. This application helps you keep track of every credit and debit in your personal network with a sleek cyberpunk aesthetic.

### "The Golden Rule of the Streets"
> *"Pag Utang Tawag Sakin Boss, Pag Singil na Tawag Sakin Bantot"*

This quote, inspired by the legendary Bumbay memes, encapsulates the fluid nature of financial respect in the matrix.

### Key Features
- **Debt Dashboard:** Real-time visualization of your total exposure and receivables.
- **Node Management:** Initialize, modify, and settle debt nodes with precision.
- **Analytics:** Data-driven insights into your financial standing across the network.
- **Cyberpunk UI:** Matte black surfaces, gold accents, and neon alerts for a high-tech feel.
- **About Section:** Integrated documentation and system versioning.

### Technical Stack
- **Kotlin** & **Coroutines** for asynchronous node processing.
- **Room Database** for persistent local storage.
- **Navigation Component** for fluid matrix traversal.
- **ViewBinding** for direct access to UI elements.
- **Material Components** for industrial-grade UI design.

### Installation
1. Clone the repository into your local terminal.
2. Build the project using Android Studio or Gradle.
3. Deploy to your handheld device.
4. System Online.
