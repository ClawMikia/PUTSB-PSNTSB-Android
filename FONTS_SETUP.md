# Fonts Setup

This project uses two Google Fonts:

## Orbitron (Cyberpunk/Tech style)
Download from: https://fonts.google.com/specimen/Orbitron

Required files → place in `app/src/main/res/font/`:
- `orbitron_regular.ttf`  (weight 400)
- `orbitron_semibold.ttf` (weight 600)
- `orbitron_bold.ttf`     (weight 700)

## Rajdhani (Body text)
Download from: https://fonts.google.com/specimen/Rajdhani

Required files → place in `app/src/main/res/font/`:
- `rajdhani_regular.ttf`  (weight 400)
- `rajdhani_medium.ttf`   (weight 500)

## Quick Setup (macOS/Linux)
```bash
# Install gfonts or download manually
# Rename files to match the above naming convention
# Place in app/src/main/res/font/
```

> **Temporary fallback:** If fonts are missing, Android will fall back to the
> default system font. The app will still build and run correctly.
